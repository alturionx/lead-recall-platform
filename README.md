# Lead Recall AI

Plataforma SaaS inteligente para recuperação automática de leads utilizando Inteligência Artificial, processamento orientado a eventos e motor de matching para geração de oportunidades comerciais.

## Arquitetura

O projeto é dividido nos seguintes serviços:

- Frontend Web
- Backend API
- Banco de Dados
- Serviço de Inteligência Artificial
- Integração WhatsApp

## Execução

A aplicação será executada utilizando Docker Compose.
